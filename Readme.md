## Run

1. Clone the repository.
```bash
    git clone https://github.com/Alexis-pr/Portafolio.git
``` 
2. In your desk or ubication that clone download, open the folder.
```bash
    git cd Portafolio
```
3. write in console or in Visual Studio Code.

``` bash
    code .
```
4. Finally execute the Index.html in your favorite browser
```bash 
    open index.html
```
